*API функции:

Статьи блога:

Get-запросы:

https://localhost:7009/Home/Service - страница клиента с услугами

https://localhost:7009/Home/Project - страница клиента с проектами

https://localhost:7009/Home/Index - главная страница клиента

https://localhost:7009/Home/Blog - страница клиента блог

https://localhost:7009/Home/Contact - страница клиента с контактами

https://localhost:7009/Worker - получение всех заявок

https://localhost:7009/Worker/EditIndex/{id} - получение заявки по заданному id

https://localhost:7009/Worker/Wedo - админская страница с проектами

https://localhost:7009/Worker/About - админская страница с услугами

https://localhost:7009/Worker/Blog - админская страница с блогом

https://localhost:7009/Worker/EditBlog/1 - получение блога по заданному id



Post - запросы:

https://localhost:7009/CreateAbout - добавление\обновление данных в бд

https://localhost:7009/EditWedo/{note} - добавление\обновление данных в бд

https://localhost:7009/CreateWedo/{note} - добавление\обновление данных в бд

https://localhost:7009/CreateBlog/{file}/{blog} - добавление\обновление данных в бд


Delete-запросы:

https://localhost:7009/DeleteWedo/{id} - удаление проекта из бд по id

https://localhost:7009/DeleteBlog/{id} - удаление блога из бд по id

https://localhost:7009/DeleteAbout/{id} - удаление услугу из бд по id
